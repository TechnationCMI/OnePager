Onze "onepager" is eigenlijk een soort blog/informatie website geworden. We wilden alle stappen die we gedaan hebben, in onze website zetten. Als alle stappen in een onepager gezet zouden worden, zou het een hele lange onepager worden, dus vandaar dat we het zo hebben gedaan.

**Technieken**
- **GitHub:** GitHub hebben we gebruikt om onze code online te zetten, zodat we met meerderen eraan konden werken.
- **Gatsby:** Gatsby is een relatief nieuw frontend framework die we hebben gekozen om te gebruiken.
- **MarkDown:** Het thema die we gekozen hebben, was ontwikkeld om content zo makkelijk mogelijk erin te verwerken. Hiervoor is MarkDown gebruik. Het is een makkelijk te gebruiken beginnersvriendelijk programmeertaal waarbij je geen code hoeft te weten. Dit is voor ons optimaal, omdat we veel tekst in onze website moesten verwerken.
- **Tinypng:** Deze website hebben we gebruikt om de formaten van al onze afbeeldingen kleiner te maken. Dit zorgt ervoor dat de pagina's op onze website sneller laden en kleiner in formaat blijft.
- **Netlify:** Netlify is een git gebaseerde webflow platform. Dit zorgt ervoor dat we onze code op GitHub kunnen plaatsen en dat automatisch word doorgevoerd naar onze website die online staat.
